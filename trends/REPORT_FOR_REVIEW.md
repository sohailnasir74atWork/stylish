## 5. Recommendation — gaming & online-identity, long-tail-first, quality-gated

### Niche
A tightly-themed **gaming / online-identity name & username generator** site, built **one focused
page per specific query**, not head-term-first. These queries are too niche for authority sites
to chase, so a small focused site can realistically rank them in US/UK/AU/CA.

### Why it's the right call *for this client* (execution-risk reducer)
The client already operates in this space — **Blox Fruits and Adopt Me audiences, existing gaming
tools, generator-building and mobile-dev experience**. That means: real audience understanding,
existing properties to cross-link/seed traffic, and proven ability to ship the tooling. This is
the single biggest reason the recommendation is strong *here* and might differ for someone else.

### ⚠️ Content standard (addresses "mass-programmatic is dead")
Google's 2024+ "scaled content abuse" policy penalises templated thin pages. So **do not** spin
hundreds of keyword-swapped templates. Each page must earn its place:
- unique, hand-written intro for that specific game/identity context
- a genuinely working generator with output logic suited to that query (not the same list reskinned)
- curated, platform-specific real examples
- a unique FAQ answering that query's actual questions
- internal links into the cluster + demonstrable utility

**Launch capped at ~15–25 genuinely-unique pages**, expand only as each proves it can rank.
Quality density beats page count.

### Starter long-tail clusters (validate before building — §8)
- **Roblox** (existing-asset advantage): roblox username generator, blox fruits crew name generator, adopt me pet name ideas, roblox display name ideas, roblox group name ideas
- **Shooters:** valorant name generator, sweaty fortnite names, fortnite name symbols, gamertag generator
- **Clan / squad:** clan name generator, guild name generator, squad name ideas, tryhard names
- **Aesthetic / anime / social:** anime username generator, aesthetic username ideas, soft/cute username ideas, matching username ideas, tiktok username ideas
- **Discord (long-tail, not head):** aesthetic discord names, discord display name ideas

## 6. RPM reality + climb-the-ladder plan

Low-CPC niche (≈$0.05–0.30/click), young, ad-block-prone, single-pageview audience → a deliberate
**volume play**, chosen because it's the only top-interest cluster a new site can rank fast.
- **Phase 1 (0–6mo) — RANK:** long-tail game/identity pages; get first US/UK/AU/CA organic traffic + AdSense live.
- **Phase 2 (6–12mo) — BROADEN:** add longer-dwell content (name-ideas listicles, aesthetic font tools) → more pageviews/session → more impressions.
- **Phase 3 (12mo+) — CLIMB RPM:** once authority exists, add high-CPC **business/brand name generator** pages. Same build skill, 5–15× RPM — viable only *after* authority.

## 7. Domain — ONE domain, and how to pick it

### One or multiple? → **ONE.**
Build a single brand that hosts every long-tail page. One domain consolidates topical + domain
authority (exactly what a new site needs to rank), needs one build, and supports the phased climb.
Multiple per-game micro-domains would split authority, multiply work, and slow every page's
ranking. (Multi-domain only suits churn-and-burn affiliate plays — wrong for an authority-building
AdSense site.)

### Choosing the name
Reviewer is right that *availability ≠ brand evidence*. Pick on brand merit: broad enough to cover
all games/platforms, short, easy to spell/type, memorable.
- **Recommended: `nicknamery.com`** — broad (nicknames/usernames across any game or platform),
  short, brandable, trademark-safe (no platform name in the domain).
- **Strong alternate: `usernamery.com`** — more literally "username generator"; slightly clunkier to say.
- Backups: `coolnamelab.com`, `coolnamery.com`, `nicknameforge.com`.

**Optional 10-minute brand test** before buying: show 3–5 people in the target audience the
shortlisted names for 5 seconds each; pick the one with best recall/spelling. Reconfirm
availability at the registrar (whois-available ≠ purchased).

## 8. MANDATORY validation gate (before buying / building)

Run the **keyword validation list** (`trends/KEYWORD_VALIDATION.csv`, ~30 keywords) through:
1. **Google Keyword Planner** → absolute monthly volume (US/UK/AU/CA).
2. **Ahrefs / Semrush** → Keyword Difficulty + **DR & referring domains of the current top-ranking
   pages** (confirm they're thin/low-DR = beatable by a new site).
3. **CPC / advertiser density** per keyword (realistic RPM; also pre-checks Phase-3 business naming).
4. Reconfirm domain at registrar.

Proceed only if volume is non-trivial AND top pages are genuinely beatable.
